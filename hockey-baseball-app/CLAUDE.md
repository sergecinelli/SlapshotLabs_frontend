# SlapshotLabs Hockey & Baseball App

Angular 20 + Angular Material + TailwindCSS application for sports analytics.

## Quick Reference

- **Start dev server:** `npm start` (port 4200)
- **Build:** `npm run build` / `npm run build:prod` / `npm run build:staging`
- **Lint:** `npm run lint`
- **Format:** `npm run format` / `npm run format:check`
- **Test:** `npm test`
